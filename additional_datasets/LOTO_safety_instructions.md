# Lockout-Tagout (LOTO) Safety Standard Instructions
Document ID: SOP-LOTO-2026

## 1. Scope
These safety procedures apply to all high-energy maintenance operations at the crude extraction terminal.

## 2. Mandatory Rules for COMPRESSOR-5 and VALVE-102
- **COMPRESSOR-5 Isolation**: Before performing work order tasks on the Instrument Air Compressor (COMPRESSOR-5), technicians must turn off the main breaker switches and bleed the primary tank pressure gauge to 0 bar.
- **VALVE-102 Lockout**: The Main Feed Control Valve (VALVE-102) must be mechanically locked in the fully-closed position using padlock tag Red-03.
- **Verification**: Always double-check safety locks before dismantling actuators.
